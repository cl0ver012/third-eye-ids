Third Eye

Third Eye is a Hybrid Intrusion Detection System that uses Deep Learning to predict network and application intrusions. It has over 99.1 % accuracy in detecting latest as well as old intrusion attacks. It has been trained on the CIC IDS 2018 and CIC IDS 2017 datasets. Benchmarked and tested the AI's performance on the NSL KDD+, IDS 2018, and IDS 2017 test sets. This webpage is used to demonstrate the performance of the AI model and the data pipelines for intrusion detection by uploading the server logs or captured packets.
